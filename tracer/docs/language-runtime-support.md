# Language Runtime Support

| Language   | Static Analysis | Runtime Trace                       | Notes                              |
| ---------- | --------------- | ----------------------------------- | ---------------------------------- |
| JavaScript | Supported       | Supported                           | Current runtime tracer             |
| Python     | Supported       | Experimental/Supported behind flags | Safe TypeScript interpreter subset |
| Java       | Supported       | Experimental/Supported behind flags | Safe TypeScript interpreter subset |

* Python parser spike exists
* Python runtime trace is experimental and requires PYTHON_TRACER_ENABLED
* Java parser spike exists.
* Java runtime trace is experimental and requires JAVA_TRACER_ENABLED
